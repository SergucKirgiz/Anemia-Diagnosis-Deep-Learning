# Anemia Diagnosis with Neural Networks
This project uses a Deep Learning model to predict anemia types based on Complete Blood Count (CBC) data.

## Features
- Built with **TensorFlow/Keras**.
- Achieved **~89.5% accuracy** on test data.
- Includes data preprocessing and feature scaling using **StandardScaler**.

## How to Run
1. Install requirements: `pip install -r requirements.txt`
2. Run the main script: `python ANM.py`
